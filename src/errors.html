List of usual errors

Misformed armored text

One or more inputs have unexpected input.

Check if Public Key input contains Public Key, Private Key input contains Private Key, or message input contains message.

Error decrypting private key: Incorrect key passphrase

Passphrase does not belong to the private key.

Check for errors in the passphrase or in the private key.

Ascii armor integrity check failed

Invalid private or public key.

Check for errors in the key.

Error decrypting message: Session key decryption failed.

Message could not be decrypted with this private key.

This message was not encrypted with public key belonging to this private key.

If you have multiple private keys, check if you used the right one.
